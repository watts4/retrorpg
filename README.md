# RetroRPG

A browser-based retro RPG with dungeon exploration, turn-based combat, and tile-based movement — styled after 16-bit classic RPGs.

## Play

Run locally (see below) or deploy to any static host.

## Features

- **Tile-based overworld** — navigate a procedurally laid-out town and dungeon map
- **Dungeon exploration** — walls, paths, dungeon entrances/exits, treasure rooms, boss rooms
- **Turn-based combat** — attack, defense, HP tracking
- **Leveling system** — gain EXP, level up, increase stats
- **Inventory & equipment** — weapons, armor, accessories, consumables
- **Inn rest mechanic** — restore HP at the inn or use Inn Portal Scrolls
- **Retro UI** — pixel-style layout with Tailwind CSS

## Stack

- React (Create React App)
- Tailwind CSS
- Lucide icons

## Run Locally

```bash
npm install
npm start
```
